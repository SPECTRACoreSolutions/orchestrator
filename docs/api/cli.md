# CLI Reference

**Status**: Draft  
**Last Updated**: 2026-01-06

## Commands

### discover

Run discovery activity:

```bash
orchestrator discover "logging service"
```

### status

Check orchestrator status:

```bash
orchestrator status
```

## Configuration

Environment variables:

- `ORCHESTRATOR_LLM_URL` - LLM API endpoint
- `ORCHESTRATOR_LLM_API_KEY` - LLM API key (optional)
- `ORCHESTRATOR_LLM_MODEL` - LLM model name (optional)
